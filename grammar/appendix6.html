# 부록 6. 철자

  • - 접미사 ( -s/-es / -ing / -ed / -er / -est / -ly )
  • - 명사/동사 + -s/-es
  • - ...y로 끝나는 단어 ( baby, carry, easy 등 )
  • - ...ie로 끝나는 동사 ( die, lie, tie )

# - 접미사 ( -s/-es / -ing / -ed / -er / -est / -ly )

접미사를 붙일 때 철자가 변화하기도 한다.

# - 명사/동사 + -s/-es

단어가 -s / -ss / -sh / -ch / -x 로 끝나면 -es를 붙인다.

예문

  • bus / buses
  • miss / misses
  • wash / washes
  • match / matches
  • search / searches
  • box / boxes

예외

  • potato / potatoes
  • tomato / tomatoes
  • do / does
  • go / goes

# - ...y로 끝나는 단어 ( baby, carry, easy 등 )

자음

내용 채워져야함

모음 ( a / e / i / o / u ) 모음 + -y로 끝나는 단어에서는 y가 바뀌지 않음

예문

  • play / plays / played
  • monkey / monkeys
  • enjoy / enjoys / enjoyed
  • buy / buys

예외

  • day / daily
  • pay / paid
  • lay / laid
  • say / said

# - ...ie로 끝나는 동사 ( die, lie, tie )

-ie로 끝나는 동사에 -ing를 붙이는 경우에는 -ie 👉 y

예문

  • die / dying
  • lie / lying
  • tie / tying
