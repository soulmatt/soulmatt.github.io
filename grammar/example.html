  • 모든게 잘 될 거야. 앞으로 어떤 문제도 우리한텐 없을거야.
  • 나 배고파. 아침부터 아무것도 못 먹었어.

# 9

9. 현재완료와 과거 2( I have done, I did )

  • 어제 정말 추웠다.
  • 관국이랑 중혁이는 10분 전에 나갔다.
  • 어렸을 떄 당신은 캔디를 많이 먹었나요?
  • 어젯밤 집에 늦게 들어왓다. 난 매우 피곤해서 바로 자러 갔다.
  • 언제 당신 친구들은 여기에 왔었나요?
  • 몇 시에 당신의 일이 끝납니까?
  • 오늘 지금까지 많은 일을 하고 있다.
  • 어제 난 많은 일을 했다.
  • 이번 주는 비가 안 오고 있다.
  • 지난 주는 비가 오지 않았다.
  • 오늘 아침 사나 봤나요?( 아직 아침임, 현재완료 )
  • 오늘 오전에 사나 봤나요?( 지금은 오후 또는 저녁임, 단순과거 )
  • 최근에 대원이 본 적 있나요?( 현재완료 )
  • 일요일에 대원이 봤나요?( 단순과거 )
  • 사나가 어디있는지 나는 모른다. 난 최근 그녀를 본 적이 없다.
  • A: 일요일에 파티에서 사나있었어?
    B: 모르겠어. 난 못봤었어.( 단순과거 )
  • 1시간 동안 기다리고 있다.( 아직 기다리는 중, 현재완료 )
  • 1시간 동안 기다렸다.( 더 이상 기다리고 있지 않음, 단순과거 )
  • 대원이는 서울 살고 있다. 그는 거기서 4년 동안 살고 있다.( 현재완료 )
  • 대원이는 일산에 30년 동안 살았었다. 지금 그는 서울에 산다. ( 단순과거 )
  • 난 마라톤을 해본 적이 없다.( 지금까지 살아오는 동안, 현재완료 )
  • 난 지난 여름 마라톤을 하지 않았다.( 단순과거 )
  • [ 휴가 마지막 날 ] 정말 좋은 휴가를 보내고 있다. 난 지금까지 정말 즐기고 있다.( 현재완료 )
  • [ 휴가 끝나고 나서 ] 정말 좋은 휴가였다. 아주 즐겁게 보냈다.( 단순과거 )

  • 커서 무슨 사람이 되고 싶니?
  • 휴가기간 동안 난 많은 책을 읽을거야.

  • 내가 돌아올 때까지 여기서 기다려줘.
  • 너 다 읽으면 그 책 좀 내게 빌려줄래?
  • 어떤말도 하지마 내 남자친구가 여기 있는 동안. 그가 떠날 때까지 기다려줘.
  • 대원이한테 전화하고, 우린 저녁 먹을 수 있어.
  • 대원이한테 전화해서 내가 파티에 대해 물어볼게.
  • 끝나는대로 내가 올게.
  • 너 뭔가 먹고 나면 기분이 나아질거야.
  • 비가 많이와. 우리 나가면 젖을거야.
  • 그 일이 한번 더 일어나면 난 화가 날거야.
  • 서둘러. 우리가 안 서두르면 우린 늦을거야.
  • 이따 쇼핑하러 나갈거야. 쇼핑 가면 난 치즈 좀 살거야.
  • 이따 쇼핑갈 수도 있어. 만약 가면 난 치즈 좀 살거야.
  • 오늘 만약 비 내리면 난 안 나갈거야.
  • 오늘밤 내가 늦을거란 걱정은 마.
  • 그들이 빨리 오지 않으면 난 기다리지 않을 것입니다.

99. 형용사와 부사 2( well / fast / late, hard / hardly )

  • 유시민은 잘 알려진 작가이다.Yoosimin is a well known writer.
  • A: 오늘 어때요?
    B: 난 정말 좋습니다. 고마워요.
  • 대원이는 매우 빠른 달리는 사람이다.( 형용사 이용 )
  • 지운이는 힘든 일 하는 사람이다.( 형용사 이용 )
  • 난 늦었습니다.( 형용사 이용 )
  • 대원이는 매우 빠르게 달릴 수 있습니다.( 부사 이용 )
  • 지운이는 힘들게 일합니다.( 부사 이용 )
  • 난 오늘 아침 늦게 일어났습니다.( 부사 이용 )
  • 대원이는 파티에서 매우 친화적이지 못했습니다. 그는 내게 거의 말도 안했습니다.
  • 우린 오직 1,2번 만났습니다. 우린 서로를 거의 알지 못해요.
  • 그는 일자리를 찾기 위해 굉장히 노력했다. 그러나 그는 기회를 가지지 못했다.
  • 난 안 놀라워, 그는 일자리를 못 찾았어. 그는 한개라도 찾을 노력도 거의 하지 않았다.
  • A: 우리 돈을 얼마나 가지고 있지?
    B: 거의 없어.
  • 이 두 카메라는 아주 비슷하다. 서로 다른점이 거의 없다.
  • 시험결과들이 매우 안 좋았다. 시험에 통과한 사람들이 거의 없다.
  • 그녀는 거의 아무것도 말하지 않았다.
  • 우린 가진 돈이 거의 없었다.
  • 너 글짓기는 끔직해. 난 거의 읽을 수가 없어.
  • 난 다리가 아프다. 난 거의 걸을수가 없어요.
  • 난 자정에는 언제나 집에 있어요. 거의 나가질 않습니다.
  • 당신이 피곤한 것은 전혀 안 놀라워요. 당신은 3일동안 못 잤자나요.
  • 상황은 진지했다, 그러나 전혀 위기상황은 아니다.

# 100

100. so / such

  • 난 그 책을 좋아하지 않았다. 이야기가 아주 바보 같았어. ( so / such )
  • 난 중혁이랑 관국이를 좋아한다. 그들은 매우 재밌거든. ( so / such )
  • 아름다운 날이다. 안 그러니? 정말 따뜻하다.
  • 그를 이해하기가 어렵다. 왜나면 그는 너무 조용하게 얘기하거든.
  • 그 책 내용이 너무 좋아 난 내려놓을 수가 없었다. ( so / such )
  • 정말 피곤해서 소파에서 잠들어 버렸다. ( so )
  • 최고의 휴가였어. 우린 정말 좋은 시간들을 보냈어. ( so / such )
  • 너무 좋은 날씨여서 우린 하루종일 해변에서 시간을 보냈다.
  • 누가 내게 말해줬는데 그 집은 100년 전에 지어졌대. 난 그렇게 오래 된지 인식 못하고 있었어.
  • 6시에 일어났기에 피곤했다. 난 대게 일찍 일어나지 않는다.
  • 날이 춥기를 기대했었어. 너무 따뜻해서 놀랐었다.
  • 난 인식 못했어 그 집이 그렇게 오래된 집일줄은.
  • 너 알잖아 진심이 아닌거. 어떻게 너 그렇게 말하니?
  • 넌 'blid'란 단어 사전에서 못찾을거야. 그런 단어는 없거든.
  • 난 오랫동안 그녈 못봐서 그녀가 어떻게 생겼는지 잊어버렸다.
  • 그녀를 오랫동안 못 봤다.
  • 그렇게 먼 곳인지 몰랐었다. ( so far )
  • 그렇게 먼 길인지 몰랐었다. ( such a long way )
  • 미안해 늦었지. 차가 엄청 막혔다. ( so much / such a lot of )

# 102

102. 비교급 ( cheaper, more expensive 등 )

  • 차로 데려다 줄까? 아님 기차 타고 갈래? 차로 가는게 나을거야. 그게 저렴해. 기차 타지마. 그게 더 비싸다.
  • 차로 가는게 기차 타고 가는거 보다 저렴하다.
  • 기차로 가는게 차로 가는거보다 더 비싸다.
  • 당신은 저보다 나이가 더 많아요.
  • 당신은 저보다 더 참을성이 있어요.
  • 시험이 내가 기대한 것 보다 더 쉬웠다.
  • 시험은 내가 기대한 것 보다 더 어려웠다.
  • 조금만 빨리 걸을 수 있어?
  • 조금 더 천천히 걸을 수 있어?
  • 더 큰 차가 갖고싶다
  • 더 믿음직한 차를 가지고 싶다
  • 어젯밤 평소보다 일찍 자러 갔다
  • 요즘은 테니스를 많이 안 친다. 예전에는 더 자주 쳤다.
  • 여기 매우 시끄럽네. 우리 더 조용한 곳으로 갈까?
  • 니가 마당을 청소한 후 부터 보기 훨씬 좋다.
  • 난 그를 잘 안다. 아마 누구보다 그에 대해 잘 알거야.
  • A: 두통은 어때? 좋아졌어?
    B: 아니, 최악이야.
  • 여기서 공원까지 내가 생각한 것 보다 멀어서 오래 걷고 있다.
  • 만약 니가 소식을 더 듣게되면 내가 알게해줘.

# 104

104. 비교급( as ... as / than )

  • 동석인 겉모습 보다 나이가 많지 않습니다.
  • 쇼핑몰은 평소보다 붐비지 않았다.
  • 지운이는 그녀가 희망한 것보다 시험을 잘 보지 못했다.
  • 날씨가 오늘 좋다. 별로 안 춥다.
  • 난 너 보다 많은 사람들을 알지 못해.
  • A: 얼마 들었어? 50불? B: 아냐, 그렇게 많이 들지 않았어.
  • 덥진 않지만, 어제처럼 춥지도 않다.
  • 난 너보다 돈 적게 썻어.
  • 쇼핑몰은 평소보다 붐비지 않았다.( less than 이용 )
