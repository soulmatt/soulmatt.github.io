# 4. 현재진행과 단순현재 2 ( I am doing / I do )

  • - 현재진행으로 잘 쓰지 않는 동사들
  • - think
  • - He is selfish / He is being selfish
  • - see / hear / smell / taste
  • - look / feel

# - 현재진행으로 잘 쓰지 않는 동사들

진행형은 동작을 나타내는 동사들과 함께 사용한다. ( They are eating / it is raining 등 )
동작을 나타내지 않는 일부 동사들 ( know / like 등 ) 은 진행형으로 쓸 수 없다.

  • I know ( I am knowing )
  • They like ( They are liking )

다음 동사들은 보통 현재진행으로 잘 쓰지 않는다.
like / love / hate / want / need / prefer / know / realize / suppose / mean / understand / believe / remember / belong / fit / contain / consist / seem

예제

  • I'm hungry, I want something to eat. ( I'm wanting )
  • Do you understand what I mean?
  • He doesn't seem very happy.

# - think

믿다 believe , 의견을 가지다 have an opnion 의 의미로 쓰일 때는 진행형으로 사용 안함.

예제

  • I think Nicky is Korean, but I'm not sure. ( I'm thinking )
  • What do you think about my plan?

고려하다 consider 의 의미로 쓰일 때는 진행형이 가능.

예제

  • I'm thinking about what happend. I often think about it.
  • Daewon is thinking of quitting his job. ( = he is considering it )

# - He is selfish / He is being selfish

He's being = 그는 ...하게 행동하고 있다. ( He's behaving, He's acting )
다음 문장을 비교해보자.

예제

  • I can't understand why he's being so selfish. He isn't usually like that.
    ( being selfish = 지금 이기적으로 행동하고 있음 )
  • He never thinks about other people. He is very selfish. ( He is being )
    ( 그는 단지 지금뿐만 아니고 평소에도 이기적임 )

am/is/are + being 은 말하는 시점에서 어떤 사람의 태도나 행동을 묘사 할 때 사용한다.
그 외의 경우에는 사용하지 않음.

예제

  • It's hot today. ( It's being hot )
  • Daewon is very tired. ( is being tired )

# - see / hear / smell / taste

이 동사들은 보통 진행형이 아니라 단순현재로 사용.

예제

  • Do you see that man over there? ( Are you seeing )
  • This room smells. Let's open a window.

흔히 can + see/hear/smell/taste 라는 표현을 자주 사용.

예제

  • I can hear a strange noise. Can you hear it?

# - look / feel

단순현재와 진행형 모두 사람이 ...해 보인다 or 기분이 ...하다 라고 말할 때 사용.

예제

  • You look good today. ( = You're looking good today )
  • How do you feel? ( = How are you feeling now? )

그러나

예제

  • I usually feel tired in the morning. ( I'm usually feeling )
