# 9. 현재완료와 과거 2( I have done, I did )

  • - 과거시제
  • - 현재완료 vs 단순과거

# - 과거시제

이미 지나간 시간에 대해 말하는 경우에는 현재완료가 아니라 과거시제를 사용 해야 한다.

예문

  • It was very cold yesterday. ( has been )
  • Gwankuk and Joonghyuk went out 10 minutes ago. ( have gone )
  • Did you eat a lot of candy when you were a child? ( Have you eaten )
  • I got home late last night. I was very tired and went straight to bed. ( ~~ ~~ )

When...? 또는 What time...? 이라고 질문할 때는 단순과거 를 사용한다.

예문

  • When did your friends get here? ( have ... gotten )
  • What time did you finish work?

현재완료 vs 단순과거

  • Daewon has lost his key. He can't get into the house. ( 현재완료 )
    Daewon lost his key yesterday. He couldn't get into the house. ( 단순과거 )
  • Is Sana here or has she left? ( 현재완료 )
    When did Sana leave?

# - 현재완료 vs 단순과거

현재완료 today / this week / since 1999
단순과거 yesterday / last week / from 1999 to 2019

TIP

  • I've done a lot of work today.( 현재완료 )
  • I did a lot of work yesterday.( 단순과거 )
  • It hasn't rained this week.( 현재완료 )
  • It didn't rain last week.( 단순과거 )
  • Have you seen Sana this morning?( 아직 아침임, 현재완료 )
  • Did you see Sana this morning?( 지금은 오후 또는 저녁임, 단순과거 )
  • Have you seen Daewon recently?( 현재완료 )
  • Did you see Daewon on sunday?( 단순과거 )
  • I don't know Where Sana is. I haven't seen her.( 최근 그녀를 보지 못했음, 현재완료 )
  • A: Was Sana at the party on Sunday?
    B: I don't think so. I didn't see her.( 단순과거 )
  • We've been waiting for an hour.( 아직 기다리는 중, 현재완료 )
  • We waited for an hour.( 더 이상 기다리고 있지 않음, 단순과거 )
  • Daewon lives in Seoul. He has lived there for 4 years.( 현재완료 )
  • Daewon lived in Ilsan for 30 years. Now he lives in Seoul.( 단순과거 )
  • I have never run a marathon.( 지금까지 살아오는 동안, 현재완료 )
  • I didn't run a marathon last summer.( 단순과거 )
  • [ 휴가 마지막 날 ] It's been a really good vacation. I've really enjoyed it.( 현재완료 )
  • [ 휴가 끝나고 나서 ] It was a really good vacation. I really enjoyed it.( 단순과거 )
