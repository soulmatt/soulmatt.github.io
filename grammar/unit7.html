# - have / has + 과거분사

살펴보기

Daewon: Have you traveled a lot, Jiwoon?
  Jiwoon: Yes, I've been to lots of places.
Daewon: Really? Have you ever been to China?
  Jiwoon: Yes, I've been to China twice.
Daewon: What about India?
  Jiwoon: No, I haven't been to India.

have / has 과거분사
I / we / they / you have( = I've 등 )
he / she / it has( = he's 등 )		 traveled
been
done 등

과거분사는 대체로 -ed로 끝나지만, 많은 동사들( been / done / written 등 )이 불규칙동사이다.
불규칙동사 목록은 부록 1 참조

# - have + 현재완료형

과거에서 현재까지 계속되는 기간에 일어난 일 에 대해 말할 떄 현재완료 사용.

예문

  • Have you ever eaten caviar? (지금까지 살아오면서)
  • We've never had a car.
  • A: Have you read Hamlet?
    B: No, I haven't read any shakespeare's plays.
  • Daewon really loves that movie. He's seen it ten times!
  • What a boring movie! It's the most boring movie I've ever seen.

# - 현재까지 계속되는 기간에 대해 말할 때

recently / in the last few days / so far / since / breakfast 등

예문

  • Have you heard from Daewon recently?
  • I've met a lot of people in the last few days.
  • Everything is going well. We haven't had any problems so far.
  • I'm hungry. I haven't eaten anything since breakfast.
  • It's nice to see you again. We haven't seen each ohter for a long time.

# - today / this morning / this year

말하는 시점에도 아직 끝나지 않은 기간 중에 일어난 일 에 대해 말할 때도 현재완료를 사용한다.
9. 현재완료와 과거 2( I have done, I did ) 참고

예문

  • I've drunk four cups of coffee today.
  • Have you had a vacation this year yet?
  • I haven't seen Dongsuk this morning. Have you?
  • Daewon hasn't studied very hard this semester.

# - 경험의 횟수에 대해 말할 때

"It's the first time something has happened" 처럼 경험의 횟수에 대해 말할 때 현재완료를 사용.

예문

  • Daewon is taking a driving lesson. It's his first one. It's the first time he has driven a car.🚗( drives )
  • Yunji has lost her passport again. This is the second time has happened.( happens )
  • Hyunkyu is calling his girlfriend again. That's the third time he's called her tonight.
