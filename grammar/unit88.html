# 88. all / every / whole

  • - all / everybody / everyone
  • - all / everything
  • - every / everybody / everyone / everything
  • - whole / all
  • - every / all / whole + 시간을 나타내는 단어
  • - all the time vs every time

# - all / everybody / everyone

everybody / everyone 대신에 all 은 사용 안함.

예문

  • Everybody enjoyed the party. ( All enjoyed )

all of + us / you / them 라고 쓰지 everybody of ... 으로는 사용 안함.

예문

  • All of us enjoyed the movies. ( Everybody of us )

# - all / everything

경우에 따라 all / everything 둘 다 사용.

예문

  • I'll do all I can to help.
  • I'll do everything I can to help.

everything 은 단독으로도 사용되지만, all 은 일반적으로 단독으로 사용 안함.

예문

  • He thinks he knows everything. ( he knows all )
  • Our vacation was disaster. Everything went wrong. ( All went wrong )
  • He knows all about issues. ( all도 뒤에 about이 올 때는 단독으로 사용 가능 )

all은 또한 유일한 것 이라는 의미로도 사용. ( the only thing / things )

예문

  • All you need is love.
  • All I've eaten today is a choco milk. ( 내가 오늘 먹은 유일한 음식 )
  • All I want for christmas is you.

# - every / everybody / everyone / everything

단수이며 단수 동사를 사용.

예문

  • Every seat in the theater was taken.
  • Everybody hates chris
  • Everyone has arrived. ( have arrived )
  • Everyone has a story.

그러나 everybody / everyone 지칭할 때는 흔히 they / them / their 사용

예문

  • Everybody said they enjoyed themselves.

# - whole / all

whole = 모든, 완전한 ( 일반적으로 뒤에는 단수명사를 사용 )

예문

  • Did you read the whole book? ( 책의 일부가 아니라 처음부터 끝까지 )
  • My mom has lived her whole life in Korea. 🇰🇷
  • I was so hungry, I ate a whole package of donut. ( 한 상자 모두 )

the / my / her 등은 whole 앞에 사용. ( whole vs all )

비교

the whole way / all the way
her whole life / all her life

일반적으로 whole은 셀 수 없는 명사와 함께 사용하지 않음.

예문

  • I've spent all the money you gave me. ( the whole money )

# - every / all / whole + 시간을 나타내는 단어

every는 어떤 일이 얼마나 자주 일어나는가를 말할 때 사용.

every day / every Monday / every 10 minutes / every three weeks 등 = ...마다

  • When we were on vacation, we went to the beach every day. ( all days )
  • The bus service is very good. There's a bus every 10 minutes.
  • We don't see each other very often - about every six months.

all day / the whole day = 하루 종일

  • We spent all day / the whole day at the beach. ( all the day )
  • Daewon was very quiet. he didn't say a word all night / the whole night.

# - all the time vs every time

예문

  • They never go out. They are at home all the time. ( 항상, 내내 )
  • Every time I see you, you look different. ( 매번, ... 할 때마다 )
