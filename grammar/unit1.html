# 1. 현재진행 ( I am doing )

  • - ...ing
  • - I am doing something
  • - today / this week / this year ...
  • - 현재진행 시제를 사용하는 동사들

# - ...ing

am/is/are + ...ing 는 현재진행이다.

주어 동사 줄임표현 ...ing
I am ( = I'm ) driving
he/she/it is ( = he's ) working
we/you/they are ( = we're ) doing

예제

  • Daewon is in his car. He is on his way to work. He is driving to work. 🚙🌬

# - I am doing something

나는 ...를 하고 있다. 즉, 어떤 일이나 동작을 시작해서 아직 끝나지 않은 상황 이다.

예제

  • Please don't make so much noise. I'm trying to work. ( I try )
  • A: Where's Dongsuk?
    B: He's taking a shower
  • Let's go out now. It isn't raining any more.
  • 파티에서 Hello, daewon. Are you enjoying the party? ( Do you enjoy )
  • What's all that noise? What's going on? ( = What's happening? )

말하고 있는 순간 그 동작이 진행되고 있지 않을 수도 있다.

예제

  • I'm reading a really good book at the moment. It's about a man who... 📖
    (말하는 순간에 책을 읽고 있지는 않다. 그의 말뜻은, 그가 책을 읽고 있는 중이지만 아직 다 읽지는 않았다는 뜻이다)
  • Conan wants to work in Korea, so he's studying Korean. 🇰🇷
    (하지만 말하고 있는 순간에 한국어를 공부하고 있는 것은 아니다)
  • Some friend of mind are building their own house. They hope to finish it next summer. 🏡

# - today / this week / this year ...

현재를 포함하는 근래의 일 에 대해 말할 때 사용한다.

예제

  • A: You're working hard today.
    B: Yes, I have a lot to do.
  • The company I work for isn't doing so well this year.

# - 현재진행 시제를 사용하는 동사들

get / change / became / increase / rise / fall / grow / improve / begin / start

이 동사들을 사용하여 근래에 일어난 변화에 대해 말할 때 현재진행 시제를 사용한다.

예제

  • Is your Korean getting better? 🇰🇷 ( Does your Korean get better? )
  • The population of the world is increasing very fast. 👶👶🍼 ( increases )
  • At first I didn't like my job, but I'm begining to enjoy it now. ( I begin )
