# 112. unless / as long as / provided / providing

  • - unless
  • - as long as
  • - 미래에 대해 말할때

# - unless

unless 는 except if 로 사용할 수 있다.

예문

  • You can't go in unless ( = except if ) you are a member. ( 회원인 경우를 제외하고는 )
  • You can go in only if you are a member. ( 오직 회원인 경우에만 )

예문을 더 살펴보자.

예문

  • I'll see you tomorrow unless I have to work late. ( = except if I have to work late )
  • There are no buses to the beach. Unless you have a car, it's difficult to get there. ( = except if you have a car )
  • Daewon hates to complain. he wouldn't complain about something unless it was really bad. ( except if it was really bad )
  • We can take a taxi to the Itawon unless you'd prefer to walk. ( except if you'd prefer to walk )

unless 대신 if ... not 을 사용 할 수도 있다.

예문

  • Unless we leave now, we'll be late. ( = If we don't leave now, we'll be late )

# - as long as

as long as / so long as / provided ( that ) / providing ( that )
...하는 한 / ...라는 조건이라면 이란 의미로 사용할 수 있다.

예문

  • You can borrow my car ( as long as / so long as ) you promise not to drive too fast. ( 과속하지 않겠다고 약속한다는 조건으로 차를 빌려주겠다 )
  • Traveling by car is convenient ( provided / providing ) you have somewhere to park. ( 주차할 곳이 있다면 )
  • ( provided / providing ) the room is clean, I don't care which hotel we stay at. ( 방이 깨끗하기만 하다면 괜찮다 )

provided / providing 단어 뒤에는 that 이 붙을 수 도 있다.

# - 미래에 대해 말할때

unless / as long as / so long as / provided / providing 뒤에는 will 을 쓰지 않고 현재시제를 사용.

예문

  • I'm not going out unless it stops raining. ( unless it will stop )
  • Providing the weather is good, we're going on a picnic. ( providing the weather will be good )

더 자세한 것은 24. when + 현재시제 참조.
